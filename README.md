## 🧢 프로젝트 소개
- 여느 때보다 kbo 열풍이 불고 있는 지금, 수 많은 팬들은 `프로야구를 직관`하기 위해 먼 길을 오고 갑니다.
- 하지만 야구장 근처에 거주하고 있지 않는 이상 주변 숙박업소나 맛집을 찾는 일은 쉽지 않습니다.
- 쓰리피트는 **구단별 홈구장 주변의 숙박 업소와 맛집을 모아 손쉽게 찾아볼 수 있는 서비스**입니다.
